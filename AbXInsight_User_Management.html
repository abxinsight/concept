Role Definitions & Hierarchy

Admin

Full system access

Data Manager

Validate & edit data

Data Entry Specialist

Enter consumption data

Higher roles inherit all permissions from lower roles

Admin

Facility Administrator

Full control over the facility account, users, and all data.

Permissions

  • Manage users & roles
  • Configure facility settings
  • Manage billing & subscription
  • Export all data & reports
  • Delete data permanently
  • + All Data Manager permissions

Data Manager

Quality & Validation

Oversees data quality, validates entries, and manages the data workflow.

Permissions

  • Validate pending entries
  • Edit/correct submitted data
  • Reject entries with feedback
  • View all dashboards & reports
  • Export validated data
  • Cannot manage users

Data Entry Specialist

Data Input

Enters consumption data into the system. Entries require validation.

Permissions

  • Enter new consumption data
  • Edit own pending entries
  • View own submission history
  • Upload data files
  • Cannot edit validated data
  • Cannot view full dashboards
